# 📄 O-Level Style Registration Form (Practice Project)

This is a government-style registration form inspired by the O-Level examination form layout.  
The project is built using HTML and JavaScript for learning and practice purposes only.

---

## 🚀 Features

- Structured Registration Form Layout
- Government-style Table Design
- Permanent & Correspondence Address Section
- ✅ "Same as Permanent Address" Auto Copy Feature (JavaScript)
- Input Fields with Validation Types
- File Upload Section (Photo, Signature, Thumb Impression)
- Clean and Organized Code Structure

---

## 🛠️ Technologies Used

- HTML5
- JavaScript (DOM Manipulation)

---

## 💡 Key Functionality

✔ When the checkbox "Same as Permanent Address" is selected,  
Correspondence address fields are automatically filled.

✔ When unchecked, the fields are cleared.
